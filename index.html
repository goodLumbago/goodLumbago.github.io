Projects
Create a room, invite friends or join strangers, and communicate exclusively through Morse code signals. If you want to say something, you have to beep it out. Built-in translator decodes transmissions in real-time so you can follow along while you learn. The goal is to make Morse code feel like a skill worth picking up, not a museum exhibit.

No database: conversations exist only in memory for the duration of the session. Once you disconnect, everything is gone. No logs, no history, no trace. Fast by design, private by default.

Built on Blazor Server and SignalR; WebSockets are a natural fit for real-time signal transmission where latency actually matters.

Started as a personal tool: a searchable repository of code snippets and notes, callable with a keyboard shortcut, so you can find what you need without leaving your editor. Fuzzy search over PostgreSQL trigrams means you don't need to remember the exact phrase, "hw to cntr a dif" works.

Authentication is production-grade: JWT with token rotation and reuse detection, automatic session invalidation, per-endpoint IP-based rate limiting, and Axios interceptors with promise-based deduplication to handle token refresh without race conditions. Added because the plan was always to deploy. That part is ready, the cloud bill isn't.

Ships as both a web app and a desktop application via Electron.
